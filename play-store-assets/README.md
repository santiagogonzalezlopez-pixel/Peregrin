# Play Store assets

Assets prepared for the Google Play Console main store listing.

## Recommended listing text

App name:

```text
Peregrin Passport
```

Short description:

```text
Tu pasaporte peregrino para santuarios, rutas y recuerdos.
```

Full description:

```text
Peregrin Passport es un pasaporte digital de peregrinación católica.

Descubre santuarios, marca los lugares santos que has visitado, sigue rutas sagradas, guarda recuerdos personales y genera certificados como memoria de tu camino.

Peregrin te ayuda a conservar de forma ordenada tu vida peregrina real: países completados, visitas registradas, rutas recorridas y logros conseguidos con el tiempo.

Diseñada para peregrinos que quieren descubrir lugares sagrados y guardar su camino con belleza, sencillez y fe.
```

## Files

- `app-icon-512.png`: app icon for the Play Store listing.
- `feature-graphic-1024x500.png`: feature graphic.

- `phone-screenshots/03-routes.png`: pilgrim routes.
- `phone-screenshots/04-journal.png`: journal and sanctuary details.
- `phone-screenshots/05-certificates.png`: certificates.
